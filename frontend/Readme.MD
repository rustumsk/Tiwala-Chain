# Frontend

This directory contains the TiwalaChain web client.

## Contents

```text
frontend/
|-- tiwala-frontend/   Main Next.js application
`-- Readme.MD
```

Open `tiwala-frontend/` for the runnable project, dependencies, scripts, and environment configuration.

## Quick Start

```powershell
cd frontend/tiwala-frontend
npm install
npm run dev
```

The local app runs at `http://localhost:3000` and expects the backend API URL from `NEXT_PUBLIC_API_BASE_URL`.
