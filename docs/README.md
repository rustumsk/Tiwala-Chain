# Documentation

This directory contains product, planning, and implementation notes for TiwalaChain.

## Files

```text
docs/
|-- algorithms-overview.md                      Simple notes on the main algorithms and decision logic
|-- directory-guide.md                          What each major directory is used for
|-- program-flow.md                             End-to-end system and feature flow notes
`-- README.md
```

## How To Use This Folder

- Keep implementation plans, PRDs, research notes, and deployment notes here.
- Keep short architecture notes here when they help new contributors understand the codebase faster.
- Keep runnable code and service-specific setup instructions in each service README.
- Prefer short, dated notes when documenting decisions that may change.
- Do not store secrets, private keys, wallet seed phrases, or production credentials in docs.
